Тестовое задание Sense-Capital. TicTacToe
В качестве БД было решено использовать MSSQL. Также в папке SqlScripts находится скрипт для создания бд.
Для проверки корректности программы, а именно - метода проверки победы (VictoryCheck) - были написаны юнит тесты.

- Id -> идентификатор соответствующий определенному полю. От 1 до 9
- Value -> значения поля. При 1 - значение X, 0 - значения O

1. /api/Ttt Get -> осуществляет запрос к бд для получения состояния полей.
2. /api/Ttt Post -> осуществляет запись значения и идентификатора в бд. Проверяет порядок ввода и поля на победу.
3. /api/Ttt Delete -> осуществляет очистку полей Id и Value в бд.
